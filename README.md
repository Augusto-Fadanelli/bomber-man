## Bomber-Man

### How to Build
  * Clone the repository:
    ````
    $ git clone https://github.com/Augusto-Fadanelli/bomber-man.git
    ````

  * Run a venv:
    ````
    $ python -m venv .env
    $ . .env/bin/activate
    ````

  * Upgrade the pip:
    ````
    $ python -m pip install --upgrade pip
    ````

  * Install dependencies:
    ````
    $ pip install numpy
    $ pip install pygame
    ````

  * Run the code:
    ````
    $ python main.py
    ````
